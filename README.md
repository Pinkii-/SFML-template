SFML-template
=============

Simple template for an SFML game. Provides a simple base with basic support for scenes, configuration and input.
Includes a base scene as an example of usage, with a player using input.

Base inspired in [Towerthousand's GraphDraw](https://github.com/Towerthousand/GraphDraw).

Makefile from [mbcrawfo's GenericMakefile](https://github.com/mbcrawfo/GenericMakefile).
